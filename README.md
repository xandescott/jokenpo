# Jokenpo

Jogo Pedra Papel Tesoura (Jokenpo) — versão web para desktop e mobile.
Demo interativa que mostra as partidas, contadores de vitórias e animações.

Status
- Demo: pronta para deploy
- Licença: MIT

Conteúdo do repositório
- index.html — página principal do jogo
- css/styles.css — estilos principais
- js/main.js — lógica do jogo e interações
- assets/ — imagens e ícones usados na interface
- package.json — scripts úteis para desenvolvimento e deploy

Como rodar localmente
1. Clone o repositório:
   git clone https://github.com/xandescott/jokenpo.git
2. Abra a pasta do projeto:
   cd xandescott
3. Abra index.html no navegador ou use um servidor local:
   npm install -g serve
   serve .

Deploy
- Recomendado: GitHub Pages, Netlify ou Vercel
- Para GitHub Pages: ative Pages nas Settings e escolha a branch main
